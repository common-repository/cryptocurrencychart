== Description ==
Show actual price data and charts from 1,000+ crypto currencies in your WordPress site on any available widget area! Select from 1000+ crypto currencies to show the actual price and recent price charts. Configure multiple widgets with different crypto currencies, charts and prices.

Requires a **free** API account at [CryptoCurrencyChart](https://www.cryptocurrencychart.com/api/documentation) to retrieve price and coin data.


== Installation ==
1. Sign up for a **free** API account at [CryptoCurrencyChart](https://www.cryptocurrencychart.com/subscription/signup) (up to 2,000 hits per month)
2. Set the API key and secret in the plugins settings page. It will show a message when an API key and secret could be used to successfully interact with the API.
3. Under `Appearance` > `Widgets` create a new CryptoCurrencyChart Price Widget and set its options.
4. If you want to add more price widgets repeat step 3, else you are done!
